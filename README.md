# Spotify 2023 Music Analysis

## Project Description
Inspired by my love of pop culture and the arts, I performed my own analysis of artists' music and their stats pulled from data spanning a 130 year period, largely on Jupyter Notebook using Python and SQL, as well as creating a dashboard for further analysis on Power BI.

## Key Questions Answered
- Which genres produced the most tracks in 2023?
- Who were the most popular artists in 2023?
- Has music gotten sadder over the decades?
- Have songs become more explicit over time?
- What old songs were still popular in 2023?
